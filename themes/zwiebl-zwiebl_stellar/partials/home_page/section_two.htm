    Taco Bell Test Kitchen
    This website allows users to make their online reservation to the Taco Bell headquarters Tesk Kitchen. The site is a single page web app on a static S3 bucket. Using AJAX and oAuth2.0 token access users are able to reserve their place through the Opentable API. Lambda and API Gateway were also integrated as a wrapper to further protect the OpenTable API from being exposed/misused.

    visit TheTacoBellTestKitchen.com
