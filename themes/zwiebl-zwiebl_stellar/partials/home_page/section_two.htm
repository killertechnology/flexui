Almost immediately after our reservation site went live, Test Kitchen by Taco Bell was completely booked. Fans even called Taco Bell headquarters and tried to beg their way in (it didn't work).

    ReactJS: single page web app
    AWS: S3 Static website, Route53, Cloudfront
    NodeJS: Lambda API proxy service
    Data I/O:
    OpenTable reservation API
    PII captured to secured EC2
    SQL Server hosted in private subnet

    Those who didn't get a seat were added to a waitlist. And the lucky ones who did, got treated to the most innovative Taco Bell food they've ever seen. The site allows users to make their online reservation from a single React.js web app running on a static S3 bucket. AJAX calls to the OpenTable reservations API are secured using AJAX and oAuth2.0 access tokens.
